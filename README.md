## Taller de Desarrollo Web – Formulario de Inscripción

Actividad académica desarrollada con HTML .

## Descripción
Formulario de inscripción para un taller de desarrollo web que recopila:
- Datos personales del estudiante
- Datos académicos
- Preferencias del taller
- Disponibilidad horaria
- Información adicional

## Tecnologías utilizadas
- HTML5 únicamente 

## Estructura
- `index.html` — Formulario principal con todos los campos requeridos
